# Lexical Analyzer Project

## 📌 Overview
This project is a simple **Lexical Analyzer (Tokenizer)** developed using HTML, CSS, and JavaScript.  
It is designed to simulate the first phase of a compiler, where source code is broken into meaningful tokens such as keywords, identifiers, operators, numbers, and symbols.

The system reads input code, analyzes it character by character, and classifies it according to lexical rules.

---

## 🚀 Features
- Tokenizes source code input
- Identifies:
  - Keywords
  - Identifiers
  - Operators
  - Numbers
  - Symbols / Punctuations
- Displays structured token output
- Simple user interface (web-based)
- Basic error handling for invalid tokens

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## ⚙️ How It Works
1. User opens the web interface
2. Code is entered or uploaded
3. Lexical analyzer processes the input
4. Tokens are generated based on rules
5. Output is displayed with token type and value
